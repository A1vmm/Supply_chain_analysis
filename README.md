# 📦 Supply Chain Analysis using Python

## 📌 Overview
This project focuses on analyzing supply chain data to uncover insights into **inventory management, demand forecasting, supplier performance, and cost optimization**.  
Using Python, I applied **data cleaning, exploratory data analysis (EDA), and visualization techniques** to identify bottlenecks and propose improvements for supply chain efficiency.

---

## ⚙️ Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn  
- **Tools:** Jupyter Notebook, Excel (for initial data exploration)  

---

## 📊 Key Features
- Data preprocessing: handling missing values, outliers, and normalization.  
- Exploratory Data Analysis (EDA): trends in demand, lead times, and supplier reliability.  
- Visualization: interactive charts for inventory levels, order fulfillment rates, and cost breakdowns.  
- Predictive modeling: demand forecasting using regression and time-series methods.  
- Business insights: recommendations for reducing costs and improving service levels.  

---

## 📈 Sample Insights
- Identified suppliers with **high lead-time variability** impacting delivery schedules.  
- Found that **inventory turnover ratio** was below industry benchmarks, suggesting overstocking.  
- Forecasted demand patterns to optimize **warehouse stocking strategies**.
